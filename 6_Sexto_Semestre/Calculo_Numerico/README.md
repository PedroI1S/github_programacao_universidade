# Calculo_Numerico

Total de arquivos de programacao neste ramo: 9

## Subpastas

- Avaliacao: 3 arquivo(s) de programacao
- Avaliacao2: 1 arquivo(s) de programacao

## O que cada codigo faz

- aprox_quadrados_minimos.m: aproxima f(x)=x por serie trigonometrica via quadrados minimos, monta sistema por integracao numerica e resolve por eliminacao de Gauss.
- Bisseccao.m: implementa o metodo da bissecao para encontrar raiz (exemplo: x^2-2), exibindo iteracoes e grafico de convergencia.
- FaatoracaoLU.m: calcula a fatoracao LU de uma matriz e exibe as matrizes L e U obtidas.
- MatrizTriangular.m: triangulariza uma matriz por eliminacao gaussiana (forma superior) para apoio na resolucao de sistemas lineares.
- Newton.m: implementa o metodo de Newton-Raphson para aproximacao de raiz com criterio de parada por erro e residuo.
