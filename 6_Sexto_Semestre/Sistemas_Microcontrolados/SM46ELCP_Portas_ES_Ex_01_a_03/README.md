# SM46ELCP_Portas_ES_Ex_01_a_03

Total de arquivos de programacao neste ramo: 5

## O que cada codigo faz

- SM46CP_Aula_01_Ex_Pisca_LED__main.c: exemplo base da TI para piscar LED em loop por software no MSP430.
- SM46CP_Portas_ES_Ex_1.c: pisca LED principal e usa interrupcao de botao (P1.3) para alternar LED secundario.
- SM46EL_Portas_ES_Ex_01.c: exercicio introdutorio de configuracao de portas com pisca de LED por delay em superloop.
- SM46EL_Portas_ES_Ex_02.c: leitura de chave sem interrupcao para escolher qual LED pisca (logica por polling).
- SM46EL_Portas_ES_Ex_03.c: versao com interrupcao de porta para alternar LED verde enquanto LED vermelho pisca no loop principal.
