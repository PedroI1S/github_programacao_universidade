# Trabalho_Final

Total de arquivos de programacao neste ramo: 2

## O que cada codigo faz

- main.c: implementa radar com MSP430, varredura por servo, leitura de distancia via HC-SR04 (capture timer) e exibicao em matriz de LED MAX7219 por SPI.
- Trabalho_comentado.c: versao comentada do mesmo projeto final, com explicacoes de configuracao de timers, portas, interrupcoes e logica de varredura.
